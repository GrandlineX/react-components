# Color Variables

> The following color variables are available for use in your application.
> These colors are used in the default theme and can be overridden by creating a custom theme.

### Default Theme

```css
  body {
    // THEME COLORS
    --glx-main-dark-blue:#1d47b0;
    --glx-main-blue:#3663d2;
    --glx-main-yellow: #ffd700;
    --glx-main-cyan: cyan;
    --glx-main-pink: pink;
    --glx-main-purple: purple;
    --glx-main-red: #ff323c;
    --glx-main-green: #4CAF50;
    --glx-main-orange: orange;
    --glx-main-secondary: #4CAF50;
    --glx-main-black:black;
    --glx-main-white: white;
    --glx-main-color: #b2fcfb;
    // ACCENT COLORS
    --glx-main-accent-01: #242930;
    --glx-main-accent-02: #272E38;
    --glx-main-accent-03: #2c333d;
    --glx-main-accent-04: #434d5b;
    --glx-main-accent-05: #505B6B;
    // COMPONENT COLORS
    --glx-main-contrast: #3663d2;
    --glx-main-button: #373737;
    --glx-main-background: #121212;
    --glx-main-background-alt: #202020;
    --glx-main-text: #f2f3f4;
    --glx-main-text-alt: black;
    --glx-main-select: #434d5b;

    // Card layout component
    --glx-tlc-card: #202020;
    --glx-tlc-card-font: white;
    // Tab layout component
    --glx-tlc-sidebar: #202020;
    --glx-tlc-sidebar-font: white;
    // Tab layout component
    --glx-tlc-header: #272E38;
    --glx-tlc-header-font: white;
    --glx-tlc-header-font-alt: #505B6B;
  }
```
